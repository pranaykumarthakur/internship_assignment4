# Conversly.ai Candidate Review Dashboard

A high-performance, interactive React dashboard built for technical recruiters to evaluate, score, and prioritize frontend engineering candidates at scale. 

This project was developed to fulfill the **Frontend Assignment 4** requirements, with a strong focus on modular component architecture, robust React state management, and a clean, responsive user experience.

---

## Setup and Run Instructions

Follow these step-by-step instructions to run the dashboard locally on your machine.

### 1. Prerequisites
Make sure you have Node.js and npm (Node Package Manager) installed on your computer. You can verify this by running `node -v` and `npm -v` in your terminal.

### 2. Clone the Repository
